## 💡 使用建议

1. **新手入门**: 先阅读 `00_START_HERE.md`，然后查看 `quickstart` 目录
2. **环境配置**: 参考 `setup` 目录下的文档
3. **部署上线**: 查看 `deployment` 目录
4. **问题排查**: 在 `bug_fixed_docs` 目录搜索类似问题
5. **API 开发**: 参考 `api_docs` 目录
6. **功能了解**: 查看 `features` 目录
7. **自动化测试**: 查看 `testing/MCP_CHROME_DEVTOOLS_GUIDE.md` ⭐
